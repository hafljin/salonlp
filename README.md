# Lumière Hair Salon LP

このリポジトリは美容室向けのランディングページ（LP）です。新規来店予約・再来店促進・店舗の世界観訴求を目的としたシングルページ構成で、スマホ・PC両対応のデザインとなっています。

※名前や人物などは全て架空です。

## 主な機能

### 🎨 デザイン・UI/UX
- モダンで洗練されたデザイン
- レスポンシブデザイン（スマホ・タブレット・PC対応）
- スムーズなスクロールアニメーション
- フェードインアニメーション
- ホバーエフェクト

### 📝 セクション構成
- **Hero**: インパクトのあるトップビジュアル（ファーストビュー）
- **Concept**: サロンのこだわりと世界観（コンセプト・特徴）
- **Menu**: メニュー・料金一覧
- **Campaign**: キャンペーン情報（新規追加）
- **Stylist**: スタイリスト紹介
- **Gallery**: 施術事例ギャラリー
- **Review**: お客様の声
- **FAQ**: よくある質問
- **Access**: 店舗情報（営業時間・アクセス・地図）
- **Contact**: お問い合わせフォーム（予約・問い合わせCTA）

### ✨ 新機能
- **予約モーダルフォーム**: オンライン予約機能
- **キャンペーンセクション**: 初回限定キャンペーンの表示
- **パフォーマンス最適化**: 画像の遅延読み込み、コード分割
- **アクセシビリティ向上**: ARIA属性、キーボードナビゲーション対応

## 技術スタック

- **React 19.2.1**: UIライブラリ
- **TypeScript**: 型安全性
- **Vite**: ビルドツール
- **Tailwind CSS**: スタイリング
- **Font Awesome**: アイコン

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 改善内容

### v2.0 の主な改善点

1. **コンポーネントの分離**: 保守性向上のため、各セクションを個別ファイルに分離
2. **予約機能の追加**: モーダルフォームによる予約機能を実装
3. **キャンペーンセクション**: 初回限定キャンペーンの表示機能を追加
4. **パフォーマンス最適化**: 
   - 画像の遅延読み込み
   - スムーズなアニメーション
   - コード分割
5. **アクセシビリティ向上**:
   - ARIA属性の追加
   - キーボードナビゲーション対応
   - フォーカス表示の改善
6. **UI/UX改善**:
   - スムーズスクロール
   - モーダルアニメーション
   - レスポンシブデザインの最適化
